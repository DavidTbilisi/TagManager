TagManager — thin GUI

Local-only; same tag database as tm on this machine. Enter a file path → Load tags → add or remove tags, or use Search for tag-based file lists. Check Dry-run to preview adds/removes without saving. Stop the app with Ctrl+C in the terminal where you ran tm gui. Set TAGMANAGER_GUI_ROOT to an absolute folder to only allow paths under that tree.

    API: GET /api/v1/gui/path-tags?path= (or /api/v1/files/tags?path=), POST /api/v1/gui/add-tags, POST /api/v1/gui/remove, GET /api/v1/search?tags=…&match_all=0|1. Open / or /api for a JSON route list. Stop server with Ctrl+C in the terminal.
